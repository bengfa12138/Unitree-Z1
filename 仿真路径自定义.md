# Z1 Custom Trajectory Examples

这个目录放了两份用于修改 Unitree Z1 仿真动作的示例代码。

## 文件说明

| 文件 | 用途 |
|---|---|
| `highcmd_development_annotated.cpp` | 官方 `highcmd_development.cpp` 的中文注释版，适合理解官方轨迹控制逻辑 |
| `z1_custom_waypoints.cpp` | 多路点自定义轨迹示例，适合直接改 `p1/p2/p3/p4` 来控制机械臂运动 |

## 应该改哪个官方文件？

如果你想改变 Z1 在仿真中的动作，优先看并修改：

```text
/root/work/unitree_z1/z1_sdk/examples/highcmd_development.cpp
```

不要优先改：

```text
z1_controller
sim_ctrl
unitree_ros
z1_description
```

原因：

```text
z1_controller / sim_ctrl 负责把 SDK 命令转发到 Gazebo
unitree_ros 负责模型和 Gazebo 仿真
z1_sdk/examples 负责生成机械臂具体动作命令
```

所以自定义轨迹主要改 `z1_sdk/examples`。

## 方法 1：直接替换官方 highcmd_development.cpp

把注释版复制到容器里的 SDK 示例目录：

```bash
cp /root/work/z1_trajectory_examples/highcmd_development_annotated.cpp \
   /root/work/unitree_z1/z1_sdk/examples/highcmd_development.cpp
```

重新编译：

```bash
cd /root/work/unitree_z1/z1_sdk/build
cmake ..
make -j$(nproc)
```

运行：

```bash
./highcmd_development
```

## 方法 2：添加新的 z1_custom_waypoints 可执行文件

把文件复制到 SDK 示例目录：

```bash
cp /root/work/z1_trajectory_examples/z1_custom_waypoints.cpp \
   /root/work/unitree_z1/z1_sdk/examples/z1_custom_waypoints.cpp
```

编辑 `CMakeLists.txt`：

```bash
cd /root/work/unitree_z1/z1_sdk
nano CMakeLists.txt
```

找到：

```cmake
set(EXAMPLES_FILES
    examples/highcmd_basic.cpp
    examples/highcmd_development.cpp
    examples/lowcmd_development.cpp
    examples/lowcmd_multirobots.cpp
)
```

加一行：

```cmake
    examples/z1_custom_waypoints.cpp
```

变成：

```cmake
set(EXAMPLES_FILES
    examples/highcmd_basic.cpp
    examples/highcmd_development.cpp
    examples/lowcmd_development.cpp
    examples/lowcmd_multirobots.cpp
    examples/z1_custom_waypoints.cpp
)
```

重新编译：

```bash
cd /root/work/unitree_z1/z1_sdk/build
cmake ..
make -j$(nproc)
```

运行：

```bash
./z1_custom_waypoints
```

## 运行顺序

保持三个终端顺序：

```text
终端 1：roslaunch unitree_gazebo z1.launch
终端 2：./sim_ctrl
终端 3：./highcmd_development 或 ./z1_custom_waypoints
```

## 轨迹怎么改？

在 `z1_custom_waypoints.cpp` 里主要改这里：

```cpp
Vec6 p1, p2, p3, p4;

p1 << 0.0, 1.2, -0.8, -0.4, 0.0, 0.0;
p2 << 0.45, 1.35, -1.05, -0.55, 0.15, 0.0;
p3 << -0.45, 1.25, -0.95, -0.45, -0.15, 0.0;
p4 << 0.0, 1.0, -0.75, -0.35, 0.0, 0.0;
```

每一行是 6 个关节角：

```text
joint1, joint2, joint3, joint4, joint5, joint6
```

单位是弧度，不是角度：

```text
30 deg  = 0.524 rad
45 deg  = 0.785 rad
90 deg  = 1.571 rad
```

每段运动时间改这里：

```cpp
std::vector<double> segmentTimes = {2.0, 2.5, 2.5, 2.0};
```

数值越大，运动越慢越平滑。
